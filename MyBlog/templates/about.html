Background information

This is a simple blog project that is used to display some of the flasks capabilities. It is built to do some simple things like allow the user to create a blog and display the blog.

Author

The author of this project is Derrick Wakhu Shibero. At the making of this project, he is currently a second year student of computer science. He has worked extensively with the flask framework and can create many things that require knowledge in flask. He is currently working on other projects that will eventually put him on the map. He also works with the django framework and is interested in machine learning and working with nosql databases like mongodb which was used in the creation of this project

Features

  • Allow user to create a blog
  • Allow user to view a blog
  • Allow user to like a view
  • Allow use to dislike a view
  • Allow user to delete a view
  • It is built entirely on flask
  • The database it uses is mongodb
